Shutting down when not in use is smart. You could also consider adding rate limiting to nginx if you want to leave it running but protect against abuse:
Copy code to clipboard
limit_req_zone $binary_remote_addr zone=demo:10m rate=10r/s;

server {
    ...
    location / {
        limit_req zone=demo burst=20;
        ...
    }
}
