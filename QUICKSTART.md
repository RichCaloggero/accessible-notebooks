# Quick Start Guide

## 30-Second Setup

### Step 1: Create Virtual Environment
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

### Step 2: Install Dependencies
```bash
cd backend
pip install -r requirements.txt
```

### Step 2: Start Backend
```bash
python server.py
```

Keep this terminal window open!

### Step 3: Open Frontend
Open `frontend/index.html` in your browser

### Step 4: Start Kernel
Click "Start Kernel" button in the browser

### Step 5: Run Code
- Navigate to Cell 1
- Press "Run Cell" button (or Ctrl+Enter)
- Listen for output announcement

## That's it!

You now have a fully functional, accessible Jupyter notebook interface.

## Quick Tips

- **Navigate cells**: Use H key (heading navigation) in screen reader
- **Run cell**: Ctrl+Enter from code textarea
- **Check status**: Kernel status indicator updates automatically
- **Restart kernel**: Use "Restart Kernel" button to clear all variables
